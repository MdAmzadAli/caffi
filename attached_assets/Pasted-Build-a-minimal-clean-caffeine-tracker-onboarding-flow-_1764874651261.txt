CALCULATION LOGIC FOR OPTIMAL CAFFEINE (implement this exactly)
Base Values

optimal = 200 (default if user skips all)

safe = 400

If user provides inputs, calculate optimal like this:

1. Weight adjustment

If weight provided:

optimal = weight_kg * 3

Cap: optimal = min(optimal, 200)

Safe limit:

safe = min(weight_kg * 6, 400)

2. Age adjustment

If age > 60 → optimal *= 0.8

If age < 18 → optimal = 80 and safe = 100

3. Gender adjustment

If female + on birth control (optional later in settings) → optimal *= 0.75

4. Sensitivity

Low → optimal *= 1.1

Medium → no change

High → optimal *= 0.5

5. Sleep Goal

Good sleep → optimal *= 0.6

Insomnia-prone → optimal *= 0.5

Normal → no change

6. Alcohol Intake

Sometimes → optimal *= 0.9

Daily → optimal *= 0.85

7. Medications

If user selects any affecting caffeine:

optimal *= 0.6

Post-Processing

Ensure: optimal = round(optimal)

Minimum allowed optimal: 50 mg

Maximum allowed optimal: 200 mg

optimal_caffeine_mg
